# 环境配置说明

## 一、Python环境配置

### 1.1 安装Python

**要求版本**: Python 3.10 或更高版本

**Windows**:
1. 访问 https://www.python.org/downloads/
2. 下载Python 3.10+安装包
3. 安装时勾选"Add Python to PATH"
4. 验证安装: `python --version`

**Linux/Mac**:
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install python3.10 python3-pip

# Mac (使用Homebrew)
brew install python@3.10
```

### 1.2 创建虚拟环境

**Windows**:
```powershell
python -m venv venv
venv\Scripts\activate
```

**Linux/Mac**:
```bash
python3 -m venv venv
source venv/bin/activate
```

### 1.3 安装依赖包

```bash
pip install -r requirements.txt
```

如果下载慢，可以使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 二、MySQL数据库配置

### 2.1 安装MySQL

**Windows**:
1. 下载MySQL安装包: https://dev.mysql.com/downloads/mysql/
2. 运行安装程序，选择"Developer Default"
3. 设置root密码并记住

**Linux**:
```bash
sudo apt update
sudo apt install mysql-server
sudo mysql_secure_installation
```

**Mac**:
```bash
brew install mysql
brew services start mysql
```

### 2.2 创建数据库

登录MySQL:
```bash
mysql -u root -p
```

执行SQL:
```sql
CREATE DATABASE bj_food_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
SHOW DATABASES;
EXIT;
```

### 2.3 配置Django数据库连接

编辑 `bj_food_analysis/settings.py`:

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'bj_food_db',              # 数据库名
        'USER': 'root',                     # 数据库用户名
        'PASSWORD': 'your_password',        # 数据库密码（修改这里）
        'HOST': 'localhost',
        'PORT': '3306',
        'OPTIONS': {'charset': 'utf8mb4'}
    }
}
```

### 2.4 执行数据库迁移

```bash
python manage.py makemigrations
python manage.py migrate
```

## 三、百度地图API配置

### 3.1 申请API Key

1. 访问 https://lbsyun.baidu.com/
2. 注册/登录账号
3. 进入控制台 → 应用管理 → 我的应用
4. 点击"创建应用"
5. 填写信息：
   - 应用名称: 北京美食分析系统
   - 应用类型: 浏览器端
   - 白名单: `http://127.0.0.1:8000/*` 和 `http://localhost:8000/*`
6. 创建后复制AK

### 3.2 配置API Key

编辑 `bj_food_analysis/settings.py`:

```python
BAIDU_MAP_AK = 'your_baidu_map_api_key_here'
```

或使用环境变量:

**Windows PowerShell**:
```powershell
$env:BAIDU_MAP_AK="your_api_key"
```

**Linux/Mac**:
```bash
export BAIDU_MAP_AK="your_api_key"
```

然后在settings.py中:
```python
BAIDU_MAP_AK = os.environ.get('BAIDU_MAP_AK', 'YOUR_BAIDU_MAP_AK')
```

## 四、Redis配置（可选，用于情感分析）

### 4.1 安装Redis

**Windows**:
1. 下载: https://github.com/microsoftarchive/redis/releases
2. 解压并运行redis-server.exe

**Linux**:
```bash
sudo apt install redis-server
sudo systemctl start redis
```

**Mac**:
```bash
brew install redis
brew services start redis
```

### 4.2 配置Redis连接

编辑 `bj_food_analysis/settings.py`:

```python
REDIS_HOST = 'localhost'
REDIS_PORT = 6379
REDIS_DB = 0
```

## 五、环境变量配置（推荐）

### 5.1 创建.env文件

在项目根目录创建 `.env` 文件:

```env
# Django配置
SECRET_KEY=your-secret-key-here
DEBUG=True
ALLOWED_HOSTS=localhost,127.0.0.1

# 数据库配置
DB_NAME=bj_food_db
DB_USER=root
DB_PASSWORD=your_mysql_password
DB_HOST=localhost
DB_PORT=3306

# 百度地图API
BAIDU_MAP_AK=your_baidu_map_api_key

# Redis配置（可选）
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_DB=0
```

### 5.2 安装python-decouple（可选）

```bash
pip install python-decouple
```

在settings.py中使用:
```python
from decouple import config

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': config('DB_NAME'),
        'USER': config('DB_USER'),
        'PASSWORD': config('DB_PASSWORD'),
        'HOST': config('DB_HOST', default='localhost'),
        'PORT': config('DB_PORT', default='3306'),
    }
}

BAIDU_MAP_AK = config('BAIDU_MAP_AK')
```

## 八、常见配置问题

### 问题1: PyMySQL连接错误

**错误**: `django.core.exceptions.ImproperlyConfigured: Error loading MySQLdb module`

**解决**: 在 `bj_food_analysis/__init__.py` 中添加:
```python
import pymysql
pymysql.install_as_MySQLdb()
```

### 问题2: 数据库字符集错误

**错误**: 中文乱码

**解决**: 确保数据库和表都使用utf8mb4字符集:
```sql
ALTER DATABASE bj_food_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 问题3: 静态文件404

**解决**: 
1. 检查 `STATIC_URL` 和 `STATICFILES_DIRS` 配置
2. 运行 `python manage.py collectstatic`

### 问题4: 地图API加载失败

**解决**:
1. 检查API Key是否正确
2. 检查白名单是否包含当前域名
3. 关闭VPN/代理后重试

## 九、生产环境配置建议

### 9.1 安全设置

```python
# settings.py
DEBUG = False
ALLOWED_HOSTS = ['your-domain.com', 'www.your-domain.com']
SECRET_KEY = os.environ.get('SECRET_KEY')  # 使用环境变量
```

### 9.2 使用Gunicorn

```bash
pip install gunicorn
gunicorn bj_food_analysis.wsgi:application --bind 0.0.0.0:8000
```

### 9.3 使用Nginx反向代理

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /static/ {
        alias /path/to/staticfiles/;
    }
}
```

### 9.4 HTTPS配置

使用Let's Encrypt免费SSL证书:
```bash
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d your-domain.com
```
