### 三、数据分析功能

#### 1. 受欢迎Top10店铺 (`/user/top10_popular/`)
- 按评论数和评分综合排序
- 显示排名、店铺名称、评分、评论数、人均消费等信息
- 前三名显示特殊图标（🥇🥈🥉）

#### 2. 区县分布分析 (`/user/county_distribution/`)
- 统计各区县店铺数量
- ECharts柱状图展示
- 显示各区县平均评分和平均价格
- 详细数据表格

#### 3. 评分分布统计 (`/user/food_score/`)
- 评分分段统计（高分、中高分、中等、低分）
- ECharts饼图和柱状图展示
- 显示平均分、最高分、最低分

#### 4. 人均消费分析 (`/user/price_analysis/`)
- 价格区间统计（0-50, 50-100, 100-200, 200-300, 300+）
- ECharts饼图展示价格分布
- 价格与评分关系散点图
- 显示平均价格、最高价格、最低价格

#### 5. 菜系类型分析 (`/user/food_type/`)
- 按菜系类型统计店铺数量
- ECharts饼图和柱状图展示
- 显示各菜系占比

#### 6. 商圈分析 (`/user/business_circle_analysis/`)
- 统计各商圈店铺数量、平均评分、平均价格
- ECharts柱状图展示Top20商圈
- 详细数据表格

### 四、数据可视化功能

#### 1. 店铺分布热力图 (`/user/shop_heatmap/`)
- 基于百度地图API的热力图
- 使用评论数作为权重
- 颜色越深表示店铺越集中

#### 2. 评论情感分析 (`/user/comment_sentiment/`)
- 情感分布统计（正面、负面、中性）
- ECharts饼图展示情感分布
- 按年份的情感趋势折线图
- 详细数据表格

#### 3. 综合数据图表 (`/user/food_chart/`)
- 菜系类型分布饼图（Top8）
- 评分分布趋势折线图
- 多维度数据展示

### 五、地图功能

#### 店铺位置地图 (`/user/shop_map/`)
- 基于百度地图API
- 显示所有店铺标记点
- 点击标记显示店铺详细信息（名称、评分、评论数、地址等）
- 支持筛选功能：
  - 按区县筛选
  - 按菜系筛选
  - 按最低评分筛选
- 信息窗口包含跳转到店铺详情的链接

### 六、推荐系统

#### 智能推荐 (`/user/shop_recommendation/`)
- 基于用户收藏的店铺类型推荐相似店铺
- 基于评分和评论数推荐热门店铺
- 如果用户没有收藏，则推荐最受欢迎的店铺
- 显示推荐理由和店铺详细信息

## 配置要求

### 百度地图API Key配置
请参考 `BAIDU_MAP_SETUP.md` 文件进行配置。

## 使用说明

1. **配置百度地图API Key**：
   - 在 `bj_food_analysis/settings.py` 中设置 `BAIDU_MAP_AK`
   - 或通过环境变量 `BAIDU_MAP_AK` 设置

2. **运行系统**：
   ```bash
   python manage.py runserver
   ```

3. **访问功能**：
   - 登录后，通过左侧导航菜单访问各项功能
   - 数据分析功能在"数据分析"菜单下
   - 数据可视化功能在"数据可视化"菜单下
   - 地图功能在"地图功能"菜单下

## 注意事项

1. 百度地图API Key需要申请并配置才能使用地图功能
2. 评论情感分析功能需要评论数据中包含 `sentiment` 字段数据
3. 所有图表使用ECharts库，需要网络连接加载CDN资源
4. 数据查询已优化，但大数据量时可能需要进一步优化
